# PatentForge v0.7 Installer — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Package PatentForge into downloadable installers for Windows (full), Mac (beta), and Linux (beta) with a Go system tray app, Node SEA binaries for the backend and feasibility services, portable Python for the three FastAPI services, and a first-run API key setup wizard.

**Architecture:** Hybrid runtime — Go tray app manages service lifecycle, NestJS backend and Express feasibility compiled to single executables via Node SEA + ncc, three Python/FastAPI services run on bundled portable Python 3.12. Backend serves frontend static files via @nestjs/serve-static (no separate frontend process). CI builds all three platform packages on tag push.

**Tech Stack:** Go 1.22+ (tray app, fyne-io/systray), Node 20 SEA + ncc (backend, feasibility), Python 3.12 portable (claim-drafter, app-generator, compliance-checker), Inno Setup (Windows), create-dmg (Mac), appimagetool (Linux), GitHub Actions (CI release workflow)

**Spec:** `docs/superpowers/specs/2026-04-02-v07-installer-design.md`

### Task 1: Add Global API Prefix to Backend

The frontend calls `/api/projects`, `/api/settings`, etc. In dev mode, Vite's proxy strips `/api` and forwards to the backend at `localhost:3000/projects`. In production (serve-static), the frontend and backend run on the same origin, so the backend must serve routes at `/api/*` to match what the frontend expects.

**Files:**
- Modify: `backend/src/main.ts`
- Modify: `backend/src/feasibility/feasibility.controller.ts` (check SSE endpoint paths)
- Test: Run existing backend tests to confirm nothing breaks

- [ ] **Step 1: Read current route paths**

Read these files to understand every route the backend exposes:
- `backend/src/main.ts`
- `backend/src/projects/projects.controller.ts`
- `backend/src/settings/settings.controller.ts`
- `backend/src/feasibility/feasibility.controller.ts`

The Vite proxy config at `frontend/vite.config.ts` strips `/api` from the path before forwarding. This means the backend currently serves at `/projects`, `/settings`, etc. We need to add a global prefix so the backend itself serves at `/api/projects`, `/api/settings`, etc.

- [ ] **Step 2: Add global prefix in main.ts**

In `backend/src/main.ts`, after `const app = await NestFactory.create(AppModule);`, add:

```typescript

```

This makes all controllers serve under `/api/*`. For example, `@Controller('projects')` now serves at `/api/projects`.

```typescript
import { Module } from '@nestjs/common';
import { ServeStaticModule } from '@nestjs/serve-static';
import { join } from 'path';
import { PrismaModule } from './prisma/prisma.module';
import { ProjectsModule } from './projects/projects.module';
import { FeasibilityModule } from './feasibility/feasibility.module';
import { SettingsModule } from './settings/settings.module';
import { PriorArtModule } from './prior-art/prior-art.module';
import { PatentDetailModule } from './patent-detail/patent-detail.module';
import { ClaimDraftModule } from './claim-draft/claim-draft.module';
import { ComplianceModule } from './compliance/compliance.module';
import { ApplicationModule } from './application/application.module';

@Module({
  imports: [
    // Serve frontend static files in production mode.
    // In dev mode (SERVE_STATIC=false or not set), this is a no-op because
    // the frontend runs on Vite's dev server at port 8080.
    ...(process.env.SERVE_STATIC !== 'false' && process.env.NODE_ENV === 'production'
      ? [ServeStaticModule.forRoot({
          rootPath: join(__dirname, '..', '..', 'frontend', 'dist'),
          exclude: ['/api/(.*)'],
        })]
      : []),
    PrismaModule,
    ProjectsModule,
    FeasibilityModule,
    SettingsModule,
    PriorArtModule,
    PatentDetailModule,
    ClaimDraftModule,
    ComplianceModule,
    ApplicationModule,
  ],
})
export class AppModule {}
```

The `rootPath` is relative to the compiled `dist/` directory — `__dirname` is `backend/dist/`, so `../../frontend/dist` reaches the frontend build output. In the installed layout, this path will be different — the tray app sets a `FRONTEND_DIST_PATH` environment variable, and we read it:

Actually, let's make it configurable:

```typescript
...(process.env.NODE_ENV === 'production'
  ? [ServeStaticModule.forRoot({
      rootPath: process.env.FRONTEND_DIST_PATH || join(__dirname, '..', '..', 'frontend', 'dist'),
      exclude: ['/api/(.*)'],
    })]
  : []),
```

- [ ] **Step 3: Implement FirstRunWizard component**

Create `frontend/src/components/FirstRunWizard.tsx`:

```typescript
import { useState } from 'react';
import { api } from '../api';
import Alert from './Alert';

interface FirstRunWizardProps {
  onComplete: (keyConfigured: boolean) => void;
}

export default function FirstRunWizard({ onComplete }: FirstRunWizardProps) {
  const [apiKey, setApiKey] = useState('');
  const [validating, setValidating] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [success, setSuccess] = useState(false);

  async function validateAndSave() {
    if (!apiKey.trim()) {
      setError('Please enter an API key.');
      return;
    }

    setValidating(true);
    setError(null);

    try {
      // Test the key against Anthropic's API directly
      const res = await fetch('https://api.anthropic.com/v1/messages', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': apiKey.trim(),
          'anthropic-version': '2023-06-01',
        },
        body: JSON.stringify({
          model: 'claude-haiku-4-5-20251001',
          max_tokens: 1,
          messages: [{ role: 'user', content: 'Hi' }],
        }),
      });

      if (!res.ok) {
        const body = await res.json().catch(() => ({}));
        if (res.status === 401) {
          setError('Invalid API key. Please check the key and try again.');
        } else if (res.status === 403) {
          setError('API key does not have permission. Check your Anthropic account.');
        } else {
          setError(body.error?.message || `API returned status ${res.status}. Try again.`);
        }
        setValidating(false);
        return;
      }

      // Key is valid — save it via the backend
      await api.settings.update({ anthropicApiKey: apiKey.trim() });
      setSuccess(true);
      setTimeout(() => onComplete(true), 1500);
    } catch (e: any) {
      if (e.message?.includes('Failed to fetch') || e.message?.includes('NetworkError')) {
        setError('Network error — check your internet connection and try again.');
      } else {
        setError(e.message || 'Validation failed. Try again.');
      }
    } finally {
      setValidating(false);
    }
  }

  return (
    <div className="min-h-screen flex items-center justify-center bg-gray-950 p-4">
      <div className="bg-gray-900 border border-gray-700 rounded-xl shadow-2xl max-w-lg w-full p-8">
        <h1 className="text-2xl font-bold text-gray-100 mb-2">
          Welcome to PatentForge!
        </h1>
        <p className="text-gray-400 mb-6">
          To analyze inventions, you need an Anthropic API key.
        </p>

        <div className="bg-gray-800 rounded-lg p-4 mb-6 text-sm text-gray-300 space-y-2">
          <p className="font-medium text-gray-200">How to get an API key:</p>
          <ol className="list-decimal ml-5 space-y-1">
            <li>
              Go to{' '}
              <a
                href="https://console.anthropic.com"
                target="_blank"
                rel="noopener noreferrer"
                className="text-blue-400 hover:text-blue-300 underline"
              >
                console.anthropic.com
              </a>
            </li>
            <li>Create a free account (or sign in)</li>
            <li>Navigate to <strong>API Keys</strong> → <strong>Create Key</strong></li>
            <li>Copy the key and paste it below</li>
          </ol>
        </div>

        <div className="mb-4">
          <label htmlFor="api-key-input" className="block text-sm font-medium text-gray-300 mb-1">
            Anthropic API Key
          </label>
          <input
            id="api-key-input"
            type="password"
            value={apiKey}
            onChange={e => setApiKey(e.target.value)}
            placeholder="sk-ant-..."
            disabled={validating || success}
            className="w-full bg-gray-800 border border-gray-600 rounded px-3 py-2 text-gray-100 placeholder-gray-500 focus:outline-none focus:border-blue-500 text-sm font-mono"
          />
        </div>

        {error && <Alert variant="error">{error}</Alert>}
        {success && <Alert variant="success">API key validated and saved! Redirecting...</Alert>}

        <div className="flex gap-3 mt-6">
          <button
            onClick={validateAndSave}
            disabled={validating || success}
            className="flex-1 px-4 py-2.5 bg-blue-600 hover:bg-blue-700 disabled:opacity-50 text-white rounded-lg font-semibold text-sm transition-colors"
          >
            {validating ? 'Validating...' : 'Validate Key'}
          </button>
          <button
            onClick={() => onComplete(false)}
            disabled={validating || success}
            className="px-4 py-2.5 bg-gray-700 hover:bg-gray-600 disabled:opacity-50 text-gray-300 rounded-lg font-medium text-sm transition-colors"
          >
            Skip for Now
          </button>
        </div>

        <p className="text-xs text-gray-500 mt-4 text-center">
          Your key is encrypted and stored locally. It is only sent to Anthropic's API.
        </p>
      </div>
    </div>
  );
}
```

Create `tray/cmd/tray/main.go`:

```go
package main

import (
	"fmt"
	"os"
	"os/exec"
	"runtime"

	"fyne.io/systray"
	"github.com/scottconverse/patentforge/tray/internal/assets"
)

var version = "0.7.0-dev"

func main() {
	systray.Run(onReady, onExit)
}

func onReady() {
	if runtime.GOOS == "windows" {
		systray.SetIcon(assets.IconICO)
	} else {
		systray.SetIcon(assets.IconPNG)
	}
	systray.SetTitle("PatentForge")
	systray.SetTooltip("PatentForge — Starting...")

	mOpen := systray.AddMenuItem("Open PatentForge", "Open in browser")
	mStatus := systray.AddMenuItem("Status: Starting...", "")
	mStatus.Disable()
	systray.AddSeparator()
	mLogs := systray.AddMenuItem("View Logs", "Open logs directory")
	mRestart := systray.AddMenuItem("Restart Services", "Restart all services")
	mAbout := systray.AddMenuItem(fmt.Sprintf("About PatentForge v%s", version), "")
	systray.AddSeparator()
	mQuit := systray.AddMenuItem("Quit", "Stop all services and exit")

	// Handle menu clicks in a goroutine
	go func() {
		for {
			select {
			case <-mOpen.ClickedCh:
				openBrowser("http://localhost:3000")
			case <-mLogs.ClickedCh:
				openFileExplorer(getLogsDir())
			case <-mRestart.ClickedCh:
				// TODO: implement in Task 7
			case <-mAbout.ClickedCh:
				openBrowser("https://github.com/scottconverse/patentforge/releases")
			case <-mQuit.ClickedCh:
				systray.Quit()
			}
		}
	}()

	// Update status after setup
	mStatus.SetTitle("Status: Running")
	systray.SetTooltip("PatentForge — Running")
}

func onExit() {
	// Cleanup will be implemented in Task 8
	fmt.Println("PatentForge shutting down...")
}

func openBrowser(url string) {
	var cmd *exec.Cmd
	switch runtime.GOOS {
	case "windows":
		cmd = exec.Command("rundll32", "url.dll,FileProtocolHandler", url)
	case "darwin":
		cmd = exec.Command("open", url)
	default:
		cmd = exec.Command("xdg-open", url)
	}
	cmd.Start()
}

func openFileExplorer(path string) {
	var cmd *exec.Cmd
	switch runtime.GOOS {
	case "windows":
		cmd = exec.Command("explorer", path)
	case "darwin":
		cmd = exec.Command("open", path)
	default:
		cmd = exec.Command("xdg-open", path)
	}
	cmd.Start()
}

func getLogsDir() string {
	exe, _ := os.Executable()
	dir := filepath.Dir(exe)
	return filepath.Join(dir, "logs")
}
```

```go
		DataDir:          filepath.Join(baseDir, "data"),

	// Create directories
	for _, dir := range []string{cfg.DataDir, cfg.LogsDir, cfg.ConfigDir} {
		if err := os.MkdirAll(dir, 0755); err != nil {
			return nil, fmt.Errorf("failed to create directory %s: %w", dir, err)
		}
	}

	// Generate .env if it doesn't exist (first run)
	if _, err := os.Stat(cfg.EnvFile); os.IsNotExist(err) {
		secret, err := generateSecret(32)

			return nil, fmt.Errorf("failed to generate secret: %w", err)

		envContent := strings.Join([]string{
			fmt.Sprintf("DATABASE_URL=%s", cfg.DatabaseURL),
			fmt.Sprintf("INTERNAL_SERVICE_SECRET=%s", cfg.ServiceSecret),
			"ALLOWED_ORIGINS=http://localhost:3000",
			"NODE_ENV=production",
			fmt.Sprintf("PORT=%d", cfg.BackendPort),
		}, "\n") + "\n"

		if err := os.WriteFile(cfg.EnvFile, []byte(envContent), 0600); err != nil {
			return nil, fmt.Errorf("failed to write .env: %w", err)
		}
	} else {
		// Read existing .env to populate config
		data, err := os.ReadFile(cfg.EnvFile)
		if err != nil {
			return nil, fmt.Errorf("failed to read .env: %w", err)
		}
		for _, line := range strings.Split(string(data), "\n") {
			parts := strings.SplitN(line, "=", 2)
			if len(parts) == 2 && parts[0] == "INTERNAL_SERVICE_SECRET" {
				cfg.ServiceSecret = parts[1]
			}
		}
	}

	return cfg, nil
}

func generateSecret(bytes int) (string, error) {
	b := make([]byte, bytes)
	if _, err := rand.Read(b); err != nil {
		return "", err
	}
```

```go
import (

	"os"
	"path/filepath"

	"syscall"

// Lock attempts to acquire an exclusive lock. Returns a cleanup function.
// On Windows, uses a lockfile with PID. On Mac/Linux, uses flock.
func Lock(baseDir string) (func(), error) {
	lockPath := filepath.Join(baseDir, "patentforge.lock")

	// Check if another instance is running
	if data, err := os.ReadFile(lockPath); err == nil {
		pid, err := strconv.Atoi(strings.TrimSpace(string(data)))
		if err == nil {
			// Check if the process is still alive
			process, err := os.FindProcess(pid)
			if err == nil {
				// On Windows, FindProcess always succeeds. Try signaling.
				err = process.Signal(syscall.Signal(0))
				if err == nil {
					return nil, fmt.Errorf("PatentForge is already running (PID %d)", pid)
				}
			}
		}
		// Stale lockfile — remove it
		os.Remove(lockPath)
	}

	// Write our PID
	if err := os.WriteFile(lockPath, []byte(strconv.Itoa(os.Getpid())), 0644); err != nil {
		return nil, fmt.Errorf("failed to create lockfile: %w", err)
	}

	cleanup := func() {
		os.Remove(lockPath)
	}

	return cleanup, nil
}
```

```go
package services

import (
	"context"
	"fmt"
	"io"
	"net/http"
	"os"
	"os/exec"
	"path/filepath"
	"sync"
	"time"
)

type ServiceStatus int

const (
	StatusStopped ServiceStatus = iota
	StatusStarting
	StatusRunning
	StatusFailed
)

func (s ServiceStatus) String() string {
	switch s {
	case StatusStopped:
		return "Stopped"
	case StatusStarting:
		return "Starting"
	case StatusRunning:
		return "Running"
	case StatusFailed:
		return "Failed"
	default:
		return "Unknown"
	}
}

type Service struct {
	Name       string
	Command    string
	Args       []string
	WorkDir    string
	Port       int
	HealthURL  string
	Env        []string
	LogFile    string

	cmd        *exec.Cmd
	status     ServiceStatus
	mu         sync.Mutex
	cancelFunc context.CancelFunc
}

func (s *Service) Start(ctx context.Context) error {
	s.mu.Lock()
	defer s.mu.Unlock()

	s.status = StatusStarting

	childCtx, cancel := context.WithCancel(ctx)
	s.cancelFunc = cancel

	s.cmd = exec.CommandContext(childCtx, s.Command, s.Args...)
	s.cmd.Dir = s.WorkDir
	s.cmd.Env = append(os.Environ(), s.Env...)

	// Redirect stdout/stderr to log file
	logDir := filepath.Dir(s.LogFile)
	os.MkdirAll(logDir, 0755)
	logF, err := os.OpenFile(s.LogFile, os.O_CREATE|os.O_WRONLY|os.O_APPEND, 0644)
	if err != nil {
		s.status = StatusFailed
		return fmt.Errorf("failed to open log file %s: %w", s.LogFile, err)
	}
	s.cmd.Stdout = logF
	s.cmd.Stderr = logF

	if err := s.cmd.Start(); err != nil {
		s.status = StatusFailed
		logF.Close()
		return fmt.Errorf("failed to start %s: %w", s.Name, err)
	}

	// Wait for port to be ready
	go func() {
		defer logF.Close()
		s.cmd.Wait()
		s.mu.Lock()
		if s.status == StatusRunning || s.status == StatusStarting {
			s.status = StatusFailed
		}
		s.mu.Unlock()
	}()

	return nil
}

func (s *Service) WaitReady(timeout time.Duration) error {
	deadline := time.Now().Add(timeout)
	for time.Now().Before(deadline) {
		resp, err := http.Get(s.HealthURL)
		if err == nil {
			io.Copy(io.Discard, resp.Body)
			resp.Body.Close()
			if resp.StatusCode < 500 {
				s.mu.Lock()
				s.status = StatusRunning
				s.mu.Unlock()
				return nil
			}
		}
		time.Sleep(500 * time.Millisecond)
	}
	return fmt.Errorf("%s did not become ready within %v", s.Name, timeout)
}

func (s *Service) Stop() {
	s.mu.Lock()
	defer s.mu.Unlock()

	if s.cancelFunc != nil {
		s.cancelFunc()
	}
	s.status = StatusStopped
}

func (s *Service) Status() ServiceStatus {
	s.mu.Lock()
	defer s.mu.Unlock()
	return s.status
}
```

```go
package services

import (
	"fmt"
	"io"
	"log"
	"net/http"
	"time"

const (
	healthInterval     = 30 * time.Second
	maxConsecFailures  = 3
	maxRestartAttempts = 3
)

type HealthMonitor struct {
	manager  *Manager
	logger   *log.Logger
	failures map[string]int // consecutive failures per service
	restarts map[string]int // restart attempts per service
	onStatusChange func(status string)
}

func NewHealthMonitor(mgr *Manager, logger *log.Logger, onStatusChange func(string)) *HealthMonitor {
	return &HealthMonitor{
		manager:        mgr,
		logger:         logger,
		failures:       make(map[string]int),
		restarts:       make(map[string]int),
		onStatusChange: onStatusChange,
	}
}

func (h *HealthMonitor) Start() {
	ticker := time.NewTicker(healthInterval)
	go func() {
		for range ticker.C {
			h.checkAll()
		}
	}()
}

func (h *HealthMonitor) checkAll() {
	for _, svc := range h.manager.services {
		if svc.Status() == StatusStopped {
			continue
		}

		healthy := h.ping(svc.HealthURL)
		if healthy {
			h.failures[svc.Name] = 0
			continue
		}

		h.failures[svc.Name]++
		h.logger.Printf("Health check failed for %s (%d consecutive)", svc.Name, h.failures[svc.Name])

		if h.failures[svc.Name] >= maxConsecFailures {
			if h.restarts[svc.Name] < maxRestartAttempts {
				h.logger.Printf("Attempting restart of %s (attempt %d/%d)", svc.Name, h.restarts[svc.Name]+1, maxRestartAttempts)
				svc.Stop()
				time.Sleep(1 * time.Second)
				if err := svc.Start(h.manager.ctx); err != nil {
					h.logger.Printf("Restart failed for %s: %v", svc.Name, err)
				}
				h.restarts[svc.Name]++
				h.failures[svc.Name] = 0
			} else {
				h.logger.Printf("%s failed after %d restart attempts", svc.Name, maxRestartAttempts)
			}
		}
	}

	status := h.manager.OverallStatus()
	h.onStatusChange(status)
}

func (h *HealthMonitor) ping(url string) bool {
	client := http.Client{Timeout: 5 * time.Second}
	resp, err := client.Get(url)
	if err != nil {
		return false
	}
	io.Copy(io.Discard, resp.Body)
	resp.Body.Close()
	return resp.StatusCode < 500
}
```

```go
package logging

import (
	"fmt"
	"io"
	"log"
	"os"
	"path/filepath"
	"sort"
	"strings"
)

const (
	maxLogSize  = 10 * 1024 * 1024 // 10MB
	maxLogFiles = 5
)

// Setup creates a logger that writes to both stdout and a rotating log file.
func Setup(logsDir string, name string) (*log.Logger, error) {
	os.MkdirAll(logsDir, 0755)
	logPath := filepath.Join(logsDir, name+".log")

	// Rotate if needed
	if info, err := os.Stat(logPath); err == nil && info.Size() > maxLogSize {
		rotate(logsDir, name)
	}

	f, err := os.OpenFile(logPath, os.O_CREATE|os.O_WRONLY|os.O_APPEND, 0644)
	if err != nil {
		return nil, fmt.Errorf("failed to open log file: %w", err)
	}

	multi := io.MultiWriter(os.Stdout, f)
	logger := log.New(multi, fmt.Sprintf("[%s] ", name), log.LstdFlags)
	return logger, nil
}

func rotate(dir, name string) {
	base := filepath.Join(dir, name)

	// Remove oldest if at max
	files := getRotatedFiles(dir, name)
	if len(files) >= maxLogFiles-1 {
		os.Remove(files[0])
	}

	// Rename current to .1, .1 to .2, etc.
	for i := maxLogFiles - 1; i >= 1; i-- {
		old := fmt.Sprintf("%s.%d.log", base, i)
		new := fmt.Sprintf("%s.%d.log", base, i+1)
		os.Rename(old, new)
	}
	os.Rename(base+".log", base+".1.log")
}

func getRotatedFiles(dir, name string) []string {
	var files []string
	entries, _ := os.ReadDir(dir)
	for _, e := range entries {
		if strings.HasPrefix(e.Name(), name+".") && strings.HasSuffix(e.Name(), ".log") && e.Name() != name+".log" {
			files = append(files, filepath.Join(dir, e.Name()))
		}
	}
	sort.Strings(files)
	return files
}
```

### Task 14: Portable Python Bundling Script

**Files:**
- Create: `scripts/bundle-python.sh`

Downloads the Python embeddable package and installs Python service dependencies into it.

- [ ] **Step 1: Create bundling script**

Create `scripts/bundle-python.sh`:

```bash
#!/bin/bash
set -e

PYTHON_VERSION="3.12.8"
PLATFORM=${1:-windows}

echo "=== Bundling Portable Python ($PLATFORM) ==="

mkdir -p runtime/python

case $PLATFORM in
    windows)
        URL="https://www.python.org/ftp/python/${PYTHON_VERSION}/python-${PYTHON_VERSION}-embed-amd64.zip"
        echo "  Downloading Python embeddable..."
        curl -L -o python-embed.zip "$URL"
        unzip -o python-embed.zip -d runtime/python/
        rm python-embed.zip

        # Enable pip in embeddable Python
        # The embeddable package doesn't include pip by default.
        # Uncomment the import site line in pythonXX._pth
        PTH_FILE=$(ls runtime/python/python*._pth)
        sed -i 's/#import site/import site/' "$PTH_FILE"

        # Install pip
        curl -L -o runtime/python/get-pip.py https://bootstrap.pypa.io/get-pip.py
        runtime/python/python.exe runtime/python/get-pip.py --no-warn-script-location
        rm runtime/python/get-pip.py
        ;;

    mac)
        # Use python.org standalone build
        URL="https://www.python.org/ftp/python/${PYTHON_VERSION}/python-${PYTHON_VERSION}-macos11.pkg"
        echo "  For Mac, use the standalone Python build from python-build-standalone project"
        echo "  or install via: brew install python@3.12"
        # This will be handled differently in CI — see release workflow
        ;;

    linux)
        # Use python-build-standalone
        echo "  For Linux, use python-build-standalone"
        # This will be handled differently in CI — see release workflow
        ;;
esac

# Install service dependencies
echo "  Installing Python service dependencies..."
PYTHON_EXE="runtime/python/python.exe"
if [ "$PLATFORM" != "windows" ]; then
    PYTHON_EXE="runtime/python/bin/python3"
fi

$PYTHON_EXE -m pip install --target runtime/python/Lib/site-packages \
    -r services/claim-drafter/requirements-frozen.txt \
    --no-warn-script-location --quiet

echo "=== Python bundling complete ==="
```

Note: We need to create `requirements-frozen.txt` files with pinned versions for each Python service, derived from the pyproject.toml dependencies. All three services share similar deps (FastAPI, LangGraph, Anthropic, uvicorn) so we install once.
