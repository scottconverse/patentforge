Prepare for your patent attorney
with AI-powered research

PatentForge is an open-source, self-hosted patent landscape research tool that helps inventors explore the patent landscape for their ideas before consulting a patent professional. Six-stage AI analysis structures your invention, discovers related prior art, and identifies the questions worth exploring with qualified counsel.

Get Started User Manual

Quick Start

git clone https://github.com/scottconverse/patentforge.git
cd patentforge
cd backend && npm install && cd ..
cd services/feasibility && npm install && cd ../..
cd frontend && npm install && cd ..
cd backend && npx prisma migrate deploy && npx prisma generate && cd ..
# Start: PatentForge.bat (Windows) or start each service manually
# Open http://localhost:8080

What PatentForge Does

Structured Technical Analysis

6-stage AI pipeline restates your invention in patent terminology, searches for related art, identifies potential issues, and organizes findings into a structured report.

Prior Art & Claims Discovery

Automated patent search via USPTO Open Data Portal with stop-word filtering and title-weighted scoring. Lazy-load actual patent claims text from the USPTO Documents API on demand.

Real-Time Streaming

Watch the AI write its findings in real time. Stage progress indicators show exactly where you are in the pipeline. Re-run individual stages without restarting.

Cost Transparency

Pre-run cost estimate with configurable cost cap. Per-stage token tracking after completion. Choose cheaper models for research stages.

Self-Hosted & Secure

Runs on your machine. API keys encrypted at rest (AES-256-GCM). Optional Bearer token auth for network deployments. Invention data stays local.

Attorney-Ready Exports

Download findings as styled HTML, Word (.docx with full formatting), or Markdown. CSV export for prior art results. Bring structured research to your first attorney meeting.

187 Automated Tests

Jest backend tests, Vitest component tests, Playwright browser E2E with screenshots and console checks, responsive viewport testing, and automated doc/version auditing. GitHub Actions CI on every push.

Prompt Integrity

SHA-256 hashes of all AI prompt files verified at startup. Drift detection warns if prompts are modified. Prompts licensed CC BY-SA 4.0 so disclaimers survive forks.

AI Claim Drafting Pipeline

After the research pipeline completes, generate patent claim drafts using a 3-agent AI pipeline. Each agent specializes in one phase of claim development.

P

Planner

Analyzes prior art & feasibility to create a claim strategy

W

Writer

Drafts 3 independent claims + dependents, capped at 20 total

E

Examiner

Reviews claims for §101/102/103/112 issues, requests revisions

Architecture

Four-service federated architecture. Each service is independently deployable. Internal services authenticated via shared secret. SSE streaming proxied through the backend. SQLite for development, PostgreSQL for Docker.

React Frontend

:8080

Invention intake form, real-time streaming output, stage progress indicators, report viewer, claims tab with editable drafts.

React 18 • TypeScript • Vite • Tailwind CSS

NestJS Backend

:3000

Project CRUD, settings management, feasibility run tracking, claim draft orchestration, SSE event forwarding, prior art search via USPTO ODP, Word export. API keys encrypted at rest. Optional Bearer token auth.

NestJS • Prisma ORM • SQLite (dev) / PostgreSQL (Docker)

Feasibility Service

:3001

6-stage sequential AI analysis pipeline. Each stage uses a markdown prompt template with the Anthropic Claude API. Streams tokens in real time via SSE.

Express • Anthropic SSE • 6 prompt templates (CC BY-SA 4.0)

Claim Drafter

:3002

3-agent claim drafting pipeline: Planner (strategy), Writer (claims), Examiner (review). LangGraph state machine with conditional revision loop. Generates up to 20 claims per draft.

Python • FastAPI • LangGraph • Anthropic SDK
Anthropic Claude API

LLM processing + web search

USPTO Open Data Portal

Patent search + claims retrieval

LiteLLM Pricing

Dynamic model cost estimation
