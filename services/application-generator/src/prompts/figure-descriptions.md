# Brief Description of the Drawings

Generate the "Brief Description of the Drawings" section for a patent application.

Since actual drawings are not available, generate descriptions for conceptual figures that would typically accompany this type of invention. Create 3-6 figure descriptions.

Format each figure description as:
FIG. 1 is a [type of diagram] illustrating [what it shows].
FIG. 2 is a [type of diagram] illustrating [what it shows].

Common figure types:
- Block diagram (system architecture)
- Flowchart (method steps)
- Schematic diagram (circuit or logical layout)
- Perspective view (physical device)
- Cross-sectional view (internal structure)
- Screenshot or user interface diagram (software)

Each description should be one sentence. The figures should cover:
1. A system overview (block diagram)
2. The main process flow (flowchart)
3. At least one detail view of a key component
4. Additional views as appropriate for the invention type

Do NOT:
- Describe what the figures contain in detail (save for Detailed Description)
- Use paragraph numbers
- Include more than one sentence per figure
